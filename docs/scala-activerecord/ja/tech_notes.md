# 技術情報

[squeryl]: https://www.squeryl.org/

## squerylライブラリとの関係

本ライブラリは ScalaのORM [squeryl] ライブラリをラップする形のものとなっております。  
squerylライブラリのスキーマ定義、カラムとのマッピング定義について、ActiveRecordパターンに則っているものを  
より簡略に定義できるよう全体的にラップしたものが本ライブラリとなります。
